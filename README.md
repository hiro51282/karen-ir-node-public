# Karen – ESP32 IR Experimental Node

ESP32上で動作する、WiFi経由の赤外線（IR）送信実験ノード。

本プロジェクトは製品実装ではなく、
「物理層（IR）とネットワーク層（WiFi）の橋渡し構造」を検証するためのログです。

---

## 背景と目的

- ESP32でのIR送信制御の確認
- HTTP経由での外部制御の実験
- プロトコルに依存しないRAW送信の検証

特定機種制御や高度な状態管理は目的に含みません。
構造検証を優先しています。

---

## 設計思想

### 1. 実験コードとしての割り切り

単一ファイル構成を採用しています。
これは再利用性よりも検証速度を優先したためです。

### 2. 責務の境界

内部構造は以下の責務を意識しています。

```

HTTP Layer
↓
Request Parsing
↓
IR Send Adapter
↓
GPIO

```

WiFi層とIR層は物理的には分離していませんが、
責務としては独立可能な構造を意識しています。

### 3. 機種依存の排除

特定メーカーAPIは含めていません。
本ノードはプロトコル実験を目的としています。

---

## エンドポイント

### RAW IR送信

```

/IR_UNKNOWN?data=9000,4500,560,560,560,1690

```

- `data` にマイクロ秒単位のパルス幅をカンマ区切りで指定
- プロトコル非依存の送信が可能

---

## 制約

- 認証なし
- LAN内利用前提
- エラーハンドリングは最小限

## ビルド

